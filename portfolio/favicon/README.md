# Favicon

このディレクトリには以下のファイルを配置してください:

- `favicon.ico` - 標準のfavicon
- `apple-touch-icon.png` - iOS用アイコン (180x180)
- `favicon-16x16.png` - 16x16のfavicon
- `favicon-32x32.png` - 32x32のfavicon

## 生成方法

オンラインツールを使用:

- https://favicon.io/
- https://realfavicongenerator.net/

または、Next.jsのApp Routerでは `app/icon.png` や `app/apple-icon.png` を配置することでも対応できます。
